# 微信小程序反编译: Vue3.0 + ant-design-vue + electron

## Run this project
```
# 1.运行 vue 项目
yarn run start:view

# 2.运行 electron 项目
yarn run start:electron
```

## build this project
```
# 1.打包 vue 项目
yarn run build:view
# 2.打包 electron 项目
yarn run build:electron
# 3.打包成 exe 程序
yarn run build
```
